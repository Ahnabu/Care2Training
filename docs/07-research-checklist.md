# Research-driven UI checklist (Gemini summary)

Source: Strategic UI/UX Modernization Report (Apr 2026).

## Must (10-day scope)
- **Hero trust signals (IDP Education inspiration)**: move key trust logos (ICEF / British Council / RCIC) to the **top hero area** for immediate credibility.
- **Services section (Apple/Iconwerk inspiration)**: replace flat 3-column layout with a **Bento Grid** layout with modern card hierarchy and hover interactions.
- **Lead generation (BrokerNotes/Lemonade inspiration)**: replace long intimidating form with a **multi-step form** (micro-commitments) to improve completion.
- **Destinations UX (StudyIn-UK inspiration)**: move from dropdown/text-only to **visual destination cards** with a modern frosted glass feel.
- **Modern styling (LHI Group + top agencies inspiration)**: introduce **vibrant gradients** for CTAs and key accents, keeping overall UI clean and premium.

## Should
- **“Scrollytelling” destination pages**: reveal stats/info smoothly as users scroll (keep motion tasteful and performance-friendly).
- **Inline social proof near form submission**: add reassurance near final submit step (e.g., “Join X+ students…”).

## Could (if time permits)
- **Short hero loop video**: dynamic background/loop (ensure mobile performance + fallback image).
