# ADR 0001: UI inspiration sources (2026)

## Decision
Adopt specific patterns proven in modern education/lead-gen sites:
- **Hero trust signals** inspired by IDP Education (logos/social proof near the top)
- **Services bento grid** inspired by Apple promo layouts + Iconwerk
- **Multi-step lead generation** inspired by BrokerNotes + Lemonade
- **Destination visual selector** inspired by StudyIn-UK
- **Vibrant gradients** inspired by LHI Group + high-end recruitment agencies

## Why
These patterns increase perceived credibility, reduce form abandonment via micro-commitments, and modernize the UI without requiring backend changes.

## Constraints
- Keep motion and glass effects performant
- Maintain accessibility (contrast, focus states, reduced motion)
