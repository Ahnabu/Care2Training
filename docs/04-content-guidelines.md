# Content guidelines (marketing UI)

## Voice & tone
- Clear, confident, supportive
- Avoid jargon; explain steps simply
- Use short paragraphs and scannable bullets

## Page patterns
- Each page should have:
  - One clear H1
  - A short value proposition
  - A primary CTA (Book appointment / Free assessment)
  - Trust signal near the top (partners, stats, testimonials)

## Duplicate copy rule
- If the same marketing text appears on 2+ pages/components, move it into `frontend/src/content/*` as a TS module and import it where needed.

## CTA hierarchy
- Primary: **Book appointment**
- Secondary: **Free assessment**
- Tertiary: **Explore destinations / Services**

## Trust signals (place early)
- Partner logos (ICEF / British Council / RCIC — if available)
- Social proof stats (visa success rate, universities, students helped)
- Office addresses + contact numbers
